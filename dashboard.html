This tool detects ransomware using 4 behavioral techniques:

🔴 Extension Detection

Monitors file creation and rename events. Flags 50+ known ransomware extensions like .locked .enc .wcry .crypto

🔴 Rapid Change Detection

Ransomware encrypts all files very fast. This tool counts file events — if 10+ changes happen in 5 seconds, it triggers a critical alert.

🟡 Entropy Analysis

Encrypted files are nearly random data — they have very high "entropy". Normal text files have low entropy. Any file with entropy > 7.5 bits/byte is flagged.

🔵 Process Monitoring

Scans running processes every 5 seconds. If any process opens files with ransomware extensions, that process is flagged by name and PID.
